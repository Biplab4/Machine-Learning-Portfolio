## 💡 Key Learning Objectives

* **Data Preprocessing Masterclass:** Managing missing values, encoding categorical data fields (`OneHotEncoder`), and implementing strategic Feature Scaling (Standardization vs Normalization).
* **Model Evaluation:** Evaluating predictive power using metrics like $R^2$ Score, Adjusted $R^2$, and Confusion Matrices.
* **Deployment Preparation:** Exporting trained mathematical pipelines into serialized files (`.pkl`) for future full-stack Django backend integration.
